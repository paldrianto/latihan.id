Bahasa yang digunakan untuk membangun dan mendesain tampilan situs web Anda. HTML menambahkan struktur, dan gaya CSS itu.

jQuery

Pustaka JavaScript yang cepat, kaya fitur, dan mudah digunakan yang menangani animasi dan permintaan Ajax.

Bahasa dinamis, tujuan umum yang sederhana dan produktif. Biasanya digunakan di seluruh dunia untuk membuat aplikasi web.

PHP

Bahasa scripting sumber terbuka yang dapat disematkan ke dalam HTML, dan sangat cocok untuk pengembangan web.
